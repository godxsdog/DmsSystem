# DMS React 前端

這是 DMS 系統的 React 前端應用程式。

## 安裝

```bash
npm install
```

## 開發

```bash
npm run dev
```

應用程式將在 http://localhost:5173 啟動。

## 環境變數

複製 `.env.example` 為 `.env` 並設定 API 基礎 URL：

```
VITE_API_BASE_URL=http://localhost:5137
```

## 建置

```bash
npm run build
```

## 功能

- 股東會明細上傳
- 公司資訊上傳
- 股票餘額上傳
