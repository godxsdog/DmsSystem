# Windows 環境快速開始指南

## 🚀 5 分鐘快速啟動

### 前置需求

- Windows 作業系統
- Visual Studio 2022
- .NET 8.0 SDK
- 正式區 SQL Server 連接權限

### 1. 取得程式碼

```powershell
# 開啟 PowerShell
cd C:\Your\Project\Path
git clone <repository-url>
cd DmsSystem
```

### 2. 還原 NuGet 套件

```powershell
dotnet restore
```

### 3. 設定資料庫連接字串

編輯 `DmsSystem.Api/appsettings.Production.json`：

```json
{
  "ConnectionStrings": {
    "DefaultConnection": "Server=正式伺服器位址;Database=DMS;User Id=正式使用者;Password=正式密碼;TrustServerCertificate=True;MultipleActiveResultSets=True"
  }
}
```

**⚠️ 重要**：請替換為正式環境的實際連接資訊

### 4. 啟動後端 API

**方法一：使用 Visual Studio 2022（推薦）**

1. 開啟 `DMS.sln`
2. 設定啟動專案為 `DmsSystem.Api`
3. 設定環境為 `Production`
4. 按 `F5` 啟動

**方法二：使用命令列**

```powershell
cd DmsSystem.Api
$env:ASPNETCORE_ENVIRONMENT="Production"
dotnet run
```

API 將在 http://localhost:5137 啟動

### 5. 啟動前端（新終端視窗）

```powershell
cd react-client
npm install
npm run dev
```

前端將在 http://localhost:5173 啟動

### 6. 查看結果

1. 打開瀏覽器訪問 http://localhost:5173
2. 點擊「資料檢視」標籤
3. 查看資料庫中的資料
