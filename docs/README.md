本文件系統提供完整的技術文件，涵蓋：

1. **系統架構**：洋蔥式架構設計和各層職責

3. **功能開發**：各功能模組的開發和使用說明
4. **測試指南**：單元測試、整合測試和手動測試
5. **資料庫操作**：資料庫配置、備份、還原和操作

## 💡 環境說明

### Windows 環境

- **資料庫**：正式區 SQL Server（已建立）
- **用途**：正式環境執行和生產部署
- **連接方式**：直接連接正式區 SQL Server
- **文件位置**：主要文件已整合在 `docs/` 根目錄

### Mac 環境

- **資料庫**：Docker SQL Server（本地測試用）
- **用途**：個人開發和測試
- **連接方式**：使用 Docker 容器運行本地 SQL Server
- **文件位置**：`MAC-DEVELOPMENT-ONLY/`
