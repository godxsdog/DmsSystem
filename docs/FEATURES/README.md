# 功能模組文件

本資料夾包含各功能模組的開發和使用文件。

## 📁 功能分類

### 1. 股東會資料管理（SHAREHOLDER-MEETING）

**功能範圍**：
- 股東會資料上傳（Excel/CSV）
- 公司資訊上傳
- 股票餘額上傳
- 資料查詢與顯示
- 報表產生

**文件位置**：`SHAREHOLDER-MEETING/`

**文件分類**：
- `BA/` - 業務分析文件
- `DEVELOPMENT/` - 程式開發文件
- `DATABASE/` - 資料庫架構文件（不含欄位定義、INSERT 語句）
- `TEST/` - 測試文件

---

### 2. 基金配息管理（DIVIDEND）

**功能範圍**：
- 配息參數設定
- 目標配息率設定
- 可分配餘額維護作業
- 配息組成資訊上傳
- 配息資訊總表轉出

**文件位置**：`DIVIDEND/`

**文件分類**：
- `BA/` - 業務分析文件
- `DEVELOPMENT/` - 程式開發文件
  - [配息功能開發指南](./DIVIDEND/DEVELOPMENT/配息功能開發指南.md)
- `DATABASE/` - 資料庫架構文件（不含欄位定義、INSERT 語句）
- `TEST/` - 測試文件

**參考文件**：
- 配息業務邏輯：`../../配息相關文件/配息業務邏輯說明 ver1.md`
- BRD 文件：`../../配息相關文件/BRD_基金配息流程_v1.md`
- 資料表清單：`../../配息相關文件/DMS5A_相關資料表清單.md`

## 🔄 開發流程

### 新增功能模組

1. **建立功能資料夾**：在 `FEATURES/` 下建立新資料夾
2. **建立分類資料夾**：建立 `BA/`, `DEVELOPMENT/`, `DATABASE/`, `TEST/`
3. **建立 README**：在功能資料夾下建立 README.md
4. **建立開發指南**：在 `DEVELOPMENT/` 下建立開發指南
5. **更新本 README**：加入新功能模組說明
6. **更新主文件索引**：更新 `../README.md`

---

## ⚠️ 重要說明

### 資料庫文件規範

**DATABASE/ 資料夾僅包含**：
- 資料庫架構說明
- 資料庫設定文件
- 連接字串設定
- 資料庫初始化腳本（不含資料）

**DATABASE/ 資料夾不包含**：
- 資料表欄位定義（column）
- INSERT INTO 語句
- 測試資料腳本
- 資料表結構查詢腳本

**這些內容應放在**：
- Mac 環境：`../MAC-DEVELOPMENT-ONLY/`（測試資料）
- Windows 環境：正式資料庫環境
